# IntelliJ IDEA #


## 檔案操作


### New File (開啟新檔)

Linux/Windows:	`alt` - `insert`

Mac:			`cmd` - `N`


### Open File (開啟檔案)

Linux/Windows:	`alt` - `insert`

Mac:			`cmd` - `o`


## 編輯

### Column Selection Mode ()

Linux/Windows:	`alt` - `shift` - `insert`

Mac:			`cmd` - `shift` - `*`

Menu:			`Edit` \ `Column Selection Mode`
