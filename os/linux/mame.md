# mame

## help

```bash
linux:~ # mame -help
```

---

## config

```bash
linux:~ # mame -showconfig
linux:~ # mame -createconfig

linux:~ # vi mame.ini
...
rompath                   "roms;$HOME/ROMs"
cheatpath                 "cheat;$HOME/cheat"
...
```

---

## run

```bash
linux:~ # mame
linux:~ # mame -window -waitvsync
```
