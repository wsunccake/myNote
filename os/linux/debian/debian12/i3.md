# i3

## install

```bash
debian:~ # apt install i3
```

---

## usage

```bash
# launch
debian:~ $ startx /usr/bin/i3

# config
debian:~ $ cat /etc/config/i32
debian:~ $ cat $HOME/.i3/config
debian:~ $ cat $HOME/.config/i3/config

debian:~ $ i3-config-wizard
```

---

## operate

```text
$mod + Enter : open terminal

---

$mod + e : split vertical / horizontal

$mod + h + Enter : open terminal in horizontal

$mod + v + Enter : open terminal in vertical

---

$mod + Shift + q : closing application window

$mod + Shift + e : exit i3

$mod + Shift + space : float window

$mod + j : left

$mod + k : down

$mod + l : up

$mod + ; : right

---

$mod + w : tab

$mod + s : stack

$mod + f : full

---

$mod + num : workspace
```
